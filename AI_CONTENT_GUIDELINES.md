# AI Content Optimization Guidelines
**For:** LGR Series Editorial Team  
**Purpose:** Maximize AI search engine visibility and citations  
**Last Updated:** February 8, 2026

## Core Principles

### 1. Opening Paragraphs (HIGH PRIORITY)
**Impact:** 67% more citations when opening paragraph directly answers the query

**Best Practice:**
- Start with a direct answer to the main question
- Use 2-3 sentences that summarize the key point
- Include the main topic/keyword in the first sentence

**Example:**
```
Local Government Reorganisation (LGR) is the process of restructuring 
local government structures in England, typically moving from a two-tier 
system (county and district councils) to single-tier unitary authorities. 
This process aims to simplify governance, reduce duplication, and improve 
service delivery efficiency.
```

**Avoid:**
- Vague introductions
- "In this article we will discuss..."
- Long background before getting to the point

### 2. Content Structure (HIGH PRIORITY)
**Impact:** 40% more citations with clear H2/H3 headings and bullet points

**Best Practice:**
- Use descriptive H2 headings for main sections
- Use H3 headings for subsections
- Include bullet points or numbered lists for key information
- Keep paragraphs concise (3-5 sentences)

**Structure Template:**
```html

<p>[Direct answer in 2-3 sentences]</p>

<h2>Key Components of LGR</h2>

<h2>How Does LGR Work?</h2>

<p>Description...</p>
```

### 3. Q&A Format
**Impact:** Q&A formats perform best in AI search results

**Best Practice:**
- Add FAQ sections to key articles
- Format questions as H3 headings
- Provide clear, concise answers
- Use FAQPage structured data (already implemented)

**Example:**
```html
<h2>Frequently Asked Questions</h2>

<h3>What is the difference between two-tier and unitary systems?</h3>
<p>In a two-tier system, responsibilities are split between county 
councils (strategic services) and district councils (local services). 
A unitary system consolidates all responsibilities into a single 
council, eliminating the split.</p>

<h3>How long does LGR take?</h3>
<p>The typical LGR process takes 2-3 years from initial proposals 
to the new unitary authority going live, including consultation, 
decision-making, shadow council formation, and transition.</p>
```

---

### 4. Bullet Points and Lists
**Impact:** Lists make content easier for AI to parse and cite

**Best Practice:**
- Use bullet points for key information
- Use numbered lists for processes or steps
- Keep list items concise (1-2 sentences)
- Use parallel structure

**Example:**
```html
<h2>Key Benefits of Unitary Authorities</h2>
<ul>
  <li><strong>Simplified governance:</strong> Single point of accountability 
  for all local services</li>
  <li><strong>Reduced duplication:</strong> Elimination of overlapping 
  responsibilities between tiers</li>
  <li><strong>Cost efficiency:</strong> Potential savings from removing 
  duplicate structures</li>
  <li><strong>Strategic capacity:</strong> Larger authorities can take on 
  more strategic responsibilities</li>
</ul>
```

---

### 5. Keywords and Context
**Best Practice:**
- Include main keywords in first paragraph
- Use natural language (avoid keyword stuffing)
- Include geographic context where relevant
- Use synonyms and related terms

**Example:**
```
Local Government Reorganisation (LGR), also known as council reform 
or local authority restructuring, involves...
```

## Examples of Well-Optimized Content

### Good Example:
```
<h1>What is Local Government Reorganisation?</h1>
<p>Local Government Reorganisation (LGR) is the process of restructuring 
local government in England, moving from two-tier systems (county and 
district councils) to single-tier unitary authorities. This transition 
aims to simplify governance, reduce duplication, and improve service 
delivery efficiency.</p>

<h2>Key Components</h2>
<ul>
  <li>Structural change: Consolidating multiple councils into unitary authorities</li>
  <li>Governance redesign: Establishing new decision-making structures</li>
  <li>Service integration: Combining services previously split across tiers</li>
</ul>
```

### Poor Example:
```
<h1>Local Government Reorganisation</h1>
<p>In recent years, there has been much discussion about the structure 
of local government in England. This article will explore various aspects 
of this topic, including historical context and current developments...</p>
[No clear answer, no structure, vague introduction]
```

## Additional Best Practices

### Author Signals
- Include author name and credentials prominently
- Add "About the Author" sections with expertise indicators
- Link to author profiles where available

### Freshness Signals
- Update publication dates when content is revised
- Include "Last updated" dates for time-sensitive content
- Reference current dates and timelines

### Geographic Context
- Include location information in structured data
- Mention specific areas/councils when relevant
- Use geographic keywords naturally
